# HTML & CSS Mini Projects

A collection of small, beginner-friendly projects built using pure HTML and CSS.  
These projects focus on UI design, animations, smooth transitions, transforms, and visual layouts.

This repo is designed to help improve CSS skills, understand design basics, and practice writing clean, beautiful, and responsive code.

Feel free to use these projects for learning, experimenting, or enhancing your front-end development journey.

## Features
- Pure HTML & CSS
- Animations & transitions
- Responsive layouts
- UI design practice
- Beginner-friendly

## Purpose
To learn, practice, and get better at front-end design through small focused mini projects.
